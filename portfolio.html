Обо мне

Я мобильный разработчик, специализирующийся на React Native (Expo) и Flutter, с сильным вниманием к чистой архитектуре, организации кода и удобству поддержки проектов.

В проектах использую:

  • Expo
  • BLoC
  • Provider
  • Clean Architecture
  • React Hooks
  • AsyncStorage
  • Deep Linking
  • API Integration

Также у меня есть опыт работы с нативными языками (Kotlin / Swift), понимание инфраструктуры Xcode и навыки деплоя в App Store, включая работу с Xcode Cloud, provisioning profiles и настройку билдов.

Уверенно владею Git и GitHub для контроля версий, включая основные операции: commit, push, merge, fetch и работу с ветками.
